---
title: "Quickstart"
description: "Make your first Workstation call in under a minute."
---

You'll need a key (see **[Get an API key](/get-api-key)**).

```bash
export KEY=sk_workstation_...

# Check your balance (free op)
curl https://<your-deploy>.convex.site/v1/balance \
  -H "Authorization: Bearer $KEY"
# → { "accountId": "acc_...", "creditsCents": 2000, "spentCents": 0 }

# Run a command in the sandbox
curl -X POST https://<your-deploy>.convex.site/v1/sandbox/exec \
  -H "Authorization: Bearer $KEY" \
  -H "Content-Type: application/json" \
  -d '{"command":"echo hello workstation"}'
# → { "stdout": "hello workstation\n", "stderr": "", "exitCode": 0 }
```

Out of credits? The next call returns `402` with a `topupUrl` — see **[Credits & billing](/credits-and-billing)**.
