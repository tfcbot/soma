---
title: "Install"
description: "Install the Workstation CLI."
---

```bash
# Using bun (recommended):
bunx workstation-cli --help

# Or the installer script (when published to GitHub Releases):
curl -fsSL https://your-landing.example.com/install.sh | bash
```

The installer pulls binaries from the operator's GitHub Releases. See
`packages/cli/install.sh` in the repo if you fork.
