---
title: "Phone"
description: "Reach people by SMS."
---

The **Phone** primitive lets an agent send SMS. The reference adapter is AgentPhone.

- `POST /v1/phone/messages {to, body}` — send an SMS, returns `{id}`.

See the [API Reference](/api-reference) for full schema, or **[Quickstart](/quickstart)** for a
worked example.
